:::snippet 接口

1. 定义接口

```typescript
interface IInterface {
  // 定义变量
  label: string;
  // 定义可选属性
  width?: number;
  // 定义只读属性
  readonly name: string;
  // 定义函数
  readName(): string;
  // 定义索引
  [propName: string]: any;
}
```

2. 实现接口

```typescript
class interfaceBaseClass implements IInterface {
  label = "labelValue";
  width = 100;
  name = "interface";
  readName() {
    return this.name;
  }
  propKey = "propValiue";
}
```

3. 接口参数限定

```typescript
let interfaceFunc = (params: IInterface): void => {};
```
